# Vehicle Rental System

**Live URL:** https://vehical-system-a2.vercel.app/



## Features

- User Registration & Login (Customer/Admin)
- JWT Authentication & Role-Based Access Control
- Vehicle Management (CRUD for Admin)
- Booking Management (Create, Update, Cancel, Auto-Return)
- Price Calculation & Vehicle Availability Management
- Real-Time Status Updates
- Responsive API Responses (Success & Error)
- PostgreSQL Database for Users, Vehicles, and Bookings
- Node.js + Express Backend with TypeScript
- Secure Password Storage (bcrypt)



## Technology Stack

- **Backend:** Node.js, Express, TypeScript  
- **Database:** PostgreSQL  
- **Authentication:** JWT, bcrypt  
- **Tools:** VS Code, Postman, Git & GitHub
